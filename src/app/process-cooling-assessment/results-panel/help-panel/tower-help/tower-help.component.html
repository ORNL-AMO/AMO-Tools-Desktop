Tower Help
Enter data related to the tower cooling system
Uses Free Cooling
Indicates if the cooling tower system uses free cooling (heat exchanger bypass). Select Yes if the system is designed to use ambient conditions for cooling when possible.
Is HEX Required
Specifies if a heat exchanger (HEX) is required for free cooling operation. Select Yes if a HEX is present or required for your system's free cooling mode.
HEX Approach Temp
The temperature difference (in °F) between the process fluid and the cooling medium in the heat exchanger during free cooling. Typical range: 5–20°F.
Number of Towers
The total number of cooling towers in the system. Enter an integer value (e.g., 1, 2, 3).
Tower Type
The supported towers are '1-Cell with 1-Speed Motor', '1-Cell with 2-Speed Motor', '2-Cell with 1-Speed Motors', '2-Cell with 2-Speed Motors', '3-Cell with 1-Speed Motors', '3-Cell with 2-Speed Motors', and 'Tower with Variable Speed Motor(s)'. By choosing one of the first six options, the number of cells per tower is automatically specified as 1, 2, or 3, where a cell is the number of identical zones (i.e. fans) operating within a cooling tower. If the seventh option is selected, an additional menu appears at the bottom of the screen to specify the number of cells per tower. It is believed that these seven options will cover the majority of cooling tower systems encountered.
Number of Fans
A cell is the number of identical, independent zones (i.e. fans) within a given cooling tower. The allowable range is 1 to 3.
Fan Speed Type
The speed control type for the tower fans. Common values: 1 = Two Speed, 2 = Variable Speed. Refer to your system's fan control method.
Tower Size Metric
The tower size can be specified by tonnage, fan hp, or declared to be Unknown. Axial fans are usually found on larger towers, and are roughly twice as efficient as centrifugal fans. The fan hp will be determined by multiplying the tower tonnage by a coefficient to arrive at a typical fan hp.
Fan Type
The type of fan used in the tower cells. This may refer to blade design, drive type, or manufacturer specification.
Tower Size
The size of the tower, either in horsepower (hp) or tonnage (tons), depending on the sizing metric. Enter a numeric value (e.g., 50 hp or 500 tons).
