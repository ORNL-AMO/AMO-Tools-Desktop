Tower Help
Enter data related to the tower cooling system
Uses Free Cooling
Indicates if the cooling tower system uses free cooling (heat exchanger bypass). Select Yes if the system is designed to use ambient conditions for cooling when possible.
Is HEX Required
Specifies if a heat exchanger (HEX) is required for free cooling operation. Select Yes if a HEX is present or required for your system's free cooling mode.
HEX Approach Temp
The temperature difference (in °F) between the process fluid and the cooling medium in the heat exchanger during free cooling. Typical range: 5–20°F.
Number of Towers
The total number of cooling towers in the system. Enter an integer value (e.g., 1, 2, 3).
Tower Type
The type or configuration of the cooling tower. This may correspond to a specific design, manufacturer, or internal code. (If unsure, consult system documentation.)
Number of Fans
The number of fans or cells per tower. This is typically set by the tower type or design. Enter an integer value.
Fan Speed Type
The speed control type for the tower fans. Common values: 1 = Two Speed, 2 = Variable Speed. Refer to your system's fan control method.
Tower Size Metric
Indicates how the tower is sized (e.g., by tonnage or fan horsepower). This is often an enumeration or code; see your system documentation for details.
Fan Type
The type of fan used in the tower cells. This may refer to blade design, drive type, or manufacturer specification.
Tower Size
The size of the tower, either in horsepower (hp) or tonnage (tons), depending on the sizing metric. Enter a numeric value (e.g., 50 hp or 500 tons).
