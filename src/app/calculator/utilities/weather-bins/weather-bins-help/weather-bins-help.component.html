Weather Bins Help
This calculator requires modified TMY3 weather data with calculated wet bulb temperatures found online. You can use the Zip Code Lookup feature if you are connected to the internet.
If you are likely to use this tool while not online, please first visit https://energyefficiency.ornl.gov/tools-training/, download the .zip file with weather data and use the Weather Station Lookup Tool to find the file name you will need based on your zip code.
If you are not located in the U.S. please use create and upload an hourly TMY weather data file with the following headers:
Date (MM/DD/YYYY), Time (HH:MM), Dry-bulb (C), RHum (%), Wet Bulb (C), Dew-point (C), Pressure (mbar), Wdir (degrees), Wspd (m/s), Lprecip depth (mm) You can also load the last file uploaded using the link below the Data File field.

Start Day
Choose the month and day you wish to start your data analysis on.
End Day
Choose the month and day you wish to end your data analysis on.
Auto-bin Parameter
Possible weather data to bin. The tool will find the minimum and maximum of the chosen parameter and automatically create bins based on the bin size chosen.
Click "Set Bins" after finalizing your auto-bin parameter and bin size.
If you wish to create your own analysis with different sized bins, click "+Add Case" for each bin you wish to create.
Bin Size
The size of the bins you wish to automatically create. The tool will find the minimum and maximum of the chosen parameter and automatically create bins based on the bin size chosen.
Click "Set Bins" after finalizing your auto-bin parameter and bin size.
If you wish to create your own analysis with different sized bins, click "+Add Case" for each bin you wish to create.
Weather Parameter
Possible weather data to bin. If creating bins manually, you can create bins with multiple parameters to find all hours that match both conditions.
Upper Bound
Highest (inclusive) value for the bin. For example, a Relative Humidtity bin with an upper bound of 95% will include values of 95% and lower, to the lower bin value.
Lower Bound
Lowest (exclusive) value for the bin. For example, a Relative Humidity bin with a lower bound of 32% will include values of 32.1% and higher, to the upper bin value.
Zip Code
Enter a valid zip code of your facility's location. This tool will find the nearest weather stations to that zip code.
