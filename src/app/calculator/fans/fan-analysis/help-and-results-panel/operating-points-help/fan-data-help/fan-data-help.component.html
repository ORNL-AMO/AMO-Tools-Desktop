Plane Information
This section is where the user more specific data about the fan system (e.g., ductwork arrangement and dimensions, the fan).
This requires the user to enter data about the overall system (any system effects to take into consideration) and any pressure drops between the fan inlet/outlet and the points of measurement. If the user can measure the static pressure of the fan inlet or outlet directly, this pressure loss is zero and values for static pressure should be entered on planes 4 or 5.
Plane 1
The first plane is the inlet of the fan.
Plane 2
Plane 2 is the fan outlet flange. If there is an evase, Plane 2 can be located either upstream or downstream of Plane 5.
Plane 3
Plane 3 is the plane used for the Pitot-static traverse for purpose of determining flow rate.
Plane 4
Plane 4 is a plane upstream of fan used to establish fan inlet Plane Static Pressure.
Plane 5
Plane 5 is a plane downstream of fan used to establish fan outlet Plane 2 Static Pressure.
Total Pressure Loss Between Planes 1 and 4
Input a positive value for the pressure loss between planes 1 and 4. This loss may be established from the frictional/dynamic losses as per ASHRAE Fundamentals ‘Duct Design’, or by some other means.
Total Pressure Loss Between Planes 2 and 5
Input a positive value for the pressure loss between planes 2 and 5. This loss may be established from the frictional/dynamic losses as per ASHRAE Fundamentals ‘Duct Design’, or by some other means.
Inlet System Effect Factor
System effect is the negative impact on fan performance resulting from connecting a duct system to the fan which has different operating conditions from when it was tested in the laboratory. The system effect at the inlet can be determined based on the use of charts and graphs, such as from Fans and Systems published by the Air Movement and Control Association (AMCA) or HVAC Systems Duct Design published by the Sheet Metal and Air Conditioning Contractors' National Association (SMACNA).
Outlet System Effect Factor
System effect is the negative impact on fan performance resulting from connecting a duct system to the fan which has different operating conditions from when it was tested in the laboratory. The system effect at the outlet can be determined based on the use of charts and graphs, such as from Fans and Systems published by the Air Movement and Control Association (AMCA) or HVAC Systems Duct Design published by the Sheet Metal and Air Conditioning Contractors' National Association (SMACNA).
Height Diameter
The inner clear Height diameter of the plane.
Width
The inner clear width of the plane
Number of Inlet Boxes
Choose either 1 or 2 depending on the number of inlet boxes. This will affect the calculation for Plane 1 planar area.
Dry Bulb Temp
Dry bulb temperature measured near the plane.
Barometric Pressure
The pressure exerted by the weight of air in the atmosphere. This defaults to the barometric pressure entered earlier in the calculator, but if there are major height changes, this can be estimated with information on the planar area elevation above sea level.
Static Pressure
Reference static pressure (relative to barometric pressure).
Pitot Tube Coefficient
If “Standard” was selected in the Pitot Tube Type Combo Box, this box will default to 1.00 as a Standard Pitot tube does not require calibration and is considered a primary instrument. If “S-Type’” was selected in the Pitot Tube Type Combo Box, the Pitot Tube Coefficient (Cp) box will default to 0.86. Typical Cp values for S-Type pitot tubes are typically in the range of 0.84 to 0.88, but the actual value depends on the specific Pitot tube being used as well as the general range of measurement the S-Type Pitot tube is being used for. Cp calibration charts for the specific Pitot tube being used should always be consulted first before inputting a value into this cell.
Number of Traverse Holes
The number of traverse points for the traverse or the number of readings per hole. This is determined from the AMCA 203 Standard, based on the size of the ductwork. The readings should be taken in the center of rectangles or concentric circles of constant area.
Number of Insertion Points
The number of insertion points used for the analysis. This is determined from the AMCA 203 Standard, based on the size of the ductwork. The readings should be taken in the center of rectangles or concentric circles of constant area.
Add Differential Readings
This is where the user enters the data from the pitot tube readings. More information on how to conduct this test can be found in the AMCA 203 standard. The rows of the table each represent all of the insertion points and the columns represent each reading along the traverse.
