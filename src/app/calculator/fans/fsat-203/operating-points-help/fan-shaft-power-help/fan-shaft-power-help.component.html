Basic Fan Info Help
This calculator lets the user determine the power, flow rate, pressure and fan efficiency at set operating points. It also helps the user perform a pitot tube analysis for the determination of flow rate and pressure.
This section sets up the basic information about the fan.
Fan Speed
The operating or nameplate speed of the fan.
Method for Establishing Motor Input Power
Select one of two options: ‘Know Motor Input Power’ or ‘Calculate Motor Input Power’ depending on if the Motor Shaft Power is known.
Drive Type
Choose among four options: ‘Direct Drive, V-Belt Drive, Notched V-Belt Drive and Synchronous Belt Drive’ depending on the drive type.
Rated Motor Power
The rated motor power.
Synchronous Speed
Synchronous speed of the motor.
Line Frequency
The main supply frequency. In North America, the standard frequency is 60 HZ. Elsewhere, 50 HZ is often the standard.
Efficiency Class
Choose the efficiency class of your motor from three options: ‘Standard Efficiency’, ‘Energy Efficient’, and ‘Premium’.
Name Plate Voltage
The motor nameplate voltage.
Full Load Amps
The motor nameplate full load amps. If this is unknown, use the link “Estimate full-load amps” to estimate.
Power Factor at Load
Motor power factor at the actual loading point of the motor.
Motor Efficiency at Load
The estimated motor efficiency at normal load conditions.
Fan Equipped with a VFD
Select ‘YES’ or ‘NO’ depending on if the fan is equipped with a VFD.
Belt Efficiency
The efficiency of the drive belt. This is set by the choice of drive type, but can be changed if known.
Motor Input Power
Motor Input Power, if known.
VFD Efficiency
If fan is equipped with a VFD, input the VFD efficiency (%).
