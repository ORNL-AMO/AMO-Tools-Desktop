Pump Curve Help
Use pump curve calculator to develop a pump curve and explore the effects of changes in head, flow, pump speed and impeller diameter.
Max Flow
The max flow rate in the pump curve graph.
Head Order
Order of the polynomial used to find head as a function of flow rate the highest power flow is raised to.
Head = C0 + C1* Flow + C2*(Flow)2 + C3*(Flow)3 has an order of 3.
Constant
Coefficient (constant) associated with flow to the zeroth power.
Head = C 0 + C1* Flow + C2*(Flow)2 + C3*(Flow)3 + C4*(Flow)4 + C5*(Flow)5 + C6*(Flow)6
Flow
Coefficient associated with flow to the first power.
Head = C0 + C 1 * Flow + C2*(Flow)2 + C3*(Flow)3 + C4*(Flow)4 + C5*(Flow)5 + C6*(Flow)6
Flow 2
Coefficient associated with flow to the second power.
Head = C0 + C1* Flow + C 2 *(Flow)2 + C3*(Flow)3 + C4*(Flow)4 + C5*(Flow)5 + C6*(Flow)6
Flow 3
Coefficient associated with flow to the third power.
Head = C0 + C1* Flow + C2*(Flow)2 + C 3 *(Flow)3 + C4*(Flow)4 + C5*(Flow)5 + C6*(Flow)6
Flow 4
Coefficient associated with flow to the fourth power.
Head = C0 + C1* Flow + C2*(Flow)2 + C3*(Flow)3 + C4 *(Flow)4 + C5*(Flow)5 + C6*(Flow)6
Flow 5
Coefficient associated with flow to the fifth power.
Head = C0 + C1* Flow + C2*(Flow)2 + C3*(Flow)3 + C4*(Flow)4 + C5 *(Flow)5 + C6*(Flow)6
Flow 5
Coefficient associated with flow to the sixth power.
Head = C0 + C1* Flow + C2*(Flow)2 + C3*(Flow)3 + C4*(Flow)4 + C5*(Flow)5 + C6 *(Flow)6
Baseline condition
The impeller diameter/speed of the system in the original pump curve.
Warning: Before reducing pump speed, it may be advisable to consult your manufacturer to avoid residence frequencies.
Diameter/Speed
Allows the user to make a new pump curve by simulating a change in pipe diameter or pump speed.
Warning: Before reducing pump speed, it may be advisable to consult your manufacturer to avoid residence frequencies.
