Fan Curve Help
Use fan curve calculator to develop a fan curve and explore the effects of changes in head, flow, fan speed and impeller diameter.
Max Flow
The max flow rate in the fan curve graph.
Constant
Coefficient (constant) associated with flow to the zeroth power.
Pressure = C 0 + C1* Flow + C2*(Flow)2 + C3*(Flow)3 + C4*(Flow)4 + C5*(Flow)5 + C6*(Flow)6
Flow
Coefficient associated with flow to the first power.
Pressure = C0 + C 1 * Flow + C2*(Flow)2 + C3*(Flow)3 + C4*(Flow)4 + C5*(Flow)5 + C6*(Flow)6
Flow 2
Coefficient associated with flow to the second power.
Pressure = C0 + C1* Flow + C 2 *(Flow)2 + C3*(Flow)3 + C4*(Flow)4 + C5*(Flow)5 + C6*(Flow)6
Flow 3
Coefficient associated with flow to the third power.
Pressure = C0 + C1* Flow + C2*(Flow)2 + C3 *(Flow)3 + C4*(Flow)4 + C5*(Flow)5 + C6*(Flow)6
Flow 4
Coefficient associated with flow to the fourth power.
Pressure = C0 + C1* Flow + C2*(Flow)2 + C3*(Flow)3 + C4 *(Flow)4 + C5*(Flow)5 + C6*(Flow)6
Flow 5
Coefficient associated with flow to the fifth power.
Pressure = C0 + C1* Flow + C2*(Flow)2 + C3*(Flow)3 + C4*(Flow)4 + C5 *(Flow)5 + C6*(Flow)6
Flow 5
Coefficient associated with flow to the sixth power.
Pressure = C0 + C1* Flow + C2*(Flow)2 + C3*(Flow)3 + C4*(Flow)4 + C5*(Flow)5 + C6 *(Flow)6
Baseline condition
The impeller diameter/speed of the system in the original fan curve.
Diameter/Speed
Allows the user to make a new pump curve by simulating a change in pipe diameter or fan speed.
Warning: Before reducing fan speed, it may be advisable to consult your manufacturer to avoid residence frequencies.
