State Point Analysis
A state point analysis is a powerful activated sludge design, operations, and troubleshooting tool. The traditional approach to a state point analysis is to develop the requisite settling flux curve using the settling parameters Vo and K. Alternatively, the requisite settling flux curve can be developed using correlations between these settling parameters and Sludge Volume Index, SVI. The SVI is performed a number of different ways:
  1. in a 1 L graduated cylinder with stirring (SVIGS)
  2. in a 1 L graduated cylinder without stirring (SVIGN)
  3. in a 2 L settlometer with stirring (SVISS)
  4. and (4) in a 2 L settlometer without stirring (SVISN)
The appropriate correlations are used to generate the flux curves in this calculator.
SVI parameter
The traditional approach to a state point analysis is to develop the requisite settling flux curve using the settling parameters Vo and K (choose “VoK”). Alternatively, the requisite settling flux curve can be developed using correlations between these settling parameters and Sludge Volume Index, SVI. The SVI is performed a number of different ways:
  1. in a 1 L graduated cylinder with stirring (SVIGS)
  2. in a 1 L graduated cylinder without stirring (SVIGN)
  3. in a 2 L settlometer with stirring (SVISS)
  4. and (4) in a 2 L settlometer without stirring (SVISN)
Sludge Volume Index
A metric to describe the ability of the sludge to settle and compact. Measured in either a graduated cylinder or settlometer and calculated as the ratio of settled sludge volume (mL/L) to Mixed Liquor Suspended Solids (g/L)
