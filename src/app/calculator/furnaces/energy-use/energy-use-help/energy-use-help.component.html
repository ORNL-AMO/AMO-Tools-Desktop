Calculation of flow rates for commonly used gases using data for an orifice type flow meter and heat input calculations when the gas is a fuel (i.e. natural gas, propane).

Gas Type

A drop down list of several gases used within fuel-fired gases. Changing this automatically changes the field “Specific gravity of gas”.

Orifice Diameter

Diameter of orifice.

Units: inch
Minimum Value Allowed: 0

Inside Pipe Diameter

Inside diameter of pipe

Units: inch
Minimum Value Allowed: 0

Orifice Pressure Drop

Pressure drop across the orifice.

Units: inch W.C.
Minimum Value Allowed: 0

Operating Time

Total time for flow / heating

Units: Hours
Minimum Value Allowed: 0

Section Type

Type of orifice. Changing this automatically changes the field “Coefficient of Discharge”

Gas Temperature

Temperature of the gas entering the orifice

Units: ℉
Minimum Value Allowed: 0

Gas Pressure

Pressure of the gas entering the orifice

Units: psig
Minimum Value Allowed: 0

Discharge Coefficient

Coefficient to take viscocity and turbulence into account when calculating flow rate.

Minimum Value Allowed: 0
Maximum Value Allowed: 1

Gas Heating Value - HHV

Higher heating value of gas (dependent on type of gas).

Units: Btu / ft3
Minimum Value Allowed: 0

Specific Gravity

Specific gravity of the gas relative to air (density of gas / density of air) at standard temperature and pressure

Minimum Value Allowed: 0
