Replace Compressor Help
This EEM models the impact of adding additional compressors to the system or removing compressors that were in the baseline.
Use the checkboxes in the tables to select which compressors to add and/or salvage/replace from the system.
Implementation Costs
Estimated cost to implement modifications.
Salvage Value
Money that can be earned by selling or recycling machinery that is removed from the system.
Trim Selection
Select the appropriate trim compressor for each day type to optimize system performance.
