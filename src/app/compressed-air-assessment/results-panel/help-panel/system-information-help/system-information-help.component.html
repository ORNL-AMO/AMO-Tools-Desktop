System Information Help
Enter data related to entire compressed air system
System Elevation
The height above mean sea level at which the compressed air system is located
Total Air Storage
The volume of air that the entire compressed air system can store, including receivers and distribution pipes.
Is a target pressure sequencer used?
Mark this box if the system air compressors are controlled by an automatic sequencer. A sequencer is a device used to bring air compressors on line according to a programmed schedule
Target Pressure
Defined pressure band that the compressors maintain by a sequenced operation.
Variance (+/-)
The defined pressure range for the target pressure that the compressors will operate around when sequenced.
CO2 Savings Help

It also helps to calculate the emissions fuel use for transportation. Select the emission source type to select the emission from combustion fuel, biofuel, refrigerant charge, purchased gases, purchased electricity from the grid. Next select the emission source and corresponding fuel type. For calculating emission from purchased electricity from the grid select Electricity under Emission Source Type.
