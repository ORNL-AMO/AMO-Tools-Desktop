Improve End Use Efficiency Help
Models airflow reductions for specific hours, achieved by reducing the amount of air required by end users
Name
Name of Improve End Use Efficiency modification.
Implementation Costs
Estimated cost to implement modifications.
Substitute Auxiliary Equipment
Average power load of replacement equipment. If airflow is reduced by replacing an end use with a electric alternative (for example a pneumatic valve with an electric actuated valve), the savings will be the difference of the decreased compressor load and the additional electrical load.
Equipment Demand
Average power required by new equipment. If airflow is reduced by replacing an end use device (for example a pneumatic valve with an electric actuated valve), that will decrease your compressor energy use, but increase your direct energy use.
Airflow Reduction Type
Can choose either fixed or variable reduction. Fixed means the same reduction will be applied to all hours, variable allows the user to set the reduction for each hour.
Airflow reduction
Amount of airflow that will be reduced
Airflow reduction (for variable)
Amount of airflow that will be reduced during that hour
