Day Type
Day types are used to group your daily operations into sets of similar operating conditions. Your day types are defined on the day types tab.
End Use Name
The identifier for a end user or group of end uses.
Description
Give a short description about the end use or group of end use.
Location
Where a end use can be found within the facility.
Average Required Pressure
The minimum pressure that the end use can operate without repercussions.
Average Airflow
The average flow the end use uses on a given day type.
Average Percent Capacity
This is a calculated value using the average airflow for the end use and the average airflow for the day type calculated using the system profile.
Regulated
Is the pressure for end use regulated, or adjusted before it reaches the end use?
Average Measured Pressure
The pressure that is actually delivered to the end use, this is a measured value.
Average Excess Pressure
The difference between the measured pressure and required pressure.
