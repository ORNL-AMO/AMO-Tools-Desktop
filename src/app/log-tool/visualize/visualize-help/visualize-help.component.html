Graph Data
Set graph data, plot options, and customize axes for the primary data series.
For Histogram plots select if data should use a standard deviation or a specific number of bins
Additional data series' can be added in the 'Other Series' tab.
Histogram Bins
The histogram will display the percentage of data points within each bin
Bins will be auto generated based on Bin Size or Number of Bins
The min and max of each bin can then be modified
Initial bin size will use the standard deviation of the data set
Primary Series
The primary x and y data series for the graph.
Additional data series' can be added in the 'Other Series' tab.
Other Data Series
Manage additional y-axis series based off the selected x-axis data from the primary data series (shown in the 'Data' tab)
A second y-axis with a different scale can added on the right hand side of the plot.
Annotations
Click the chart to select the data you want to annotate
The annotation feature may cause slow graph performance on some large datasets. MEASUR considers a large dataset to be greater than 200,000 datapoints, or roughly 7 days of log data at 3 second intervals.
Large Dataset Tips
  • Use the 'Manage Annotations' toggle to turn annotations off/on at any time (Existing annotations will remain visible and will not affect performance)
  • Narrowing the range of the graph by clicking and dragging an area before turning on annotations will increase graph speed and usability
Fill the text box with the desired annotation
  • Add <br> to create a new line
  • Wrap your text in <b></b> to bold it
  • <b>example</b> = example
  • Wrap your text in <i></i> to italicize it
  • <i>example</i> = example
Use the arrow buttons to move the location of the annotation Existing annotations can be deleted or selected to be edited from the list at the bottom of the menu
Visualize Graph
Using the graph toolbar (top-right of graph) you can:
  • Download the plot as a png
  • Change cursor types to manipulate the plot
  • Zoom in, out and scale
  • Choose method in which hover data is displayed
Click and drag the axis of the plot to change the scale
Double-click inside the graph to reset zoom orientation
Use the "+ Add Graph" tab to create a copy of the current plot
