Data Exploration Help

This module is designed to help users establish Day Types for equipment analysis incorporating multiple operating types (e.g., Compressed Air Assessments). It also provides a robust Data Visualization component to allow users to create scatter plots and histograms of their data.
Choose a .csv or .xlsx data file from your file system
  • Select your data file and the corresponding header row. If there are unused rows in between the header row and the start of the data then you will need to manually remove those rows manually before processing.
    - If you selected a .xlsx data file, each sheet of the workbook will be treated as a .csv data file. You can select the worksheet from which the data will be used.
  • There is no restrictions on the column names (length/special characters). Columns can be renamed within the Data Exploration tool if desired.
  • There is NO required date format (e.g., MM/DD/YY vs. DD-MM-YYYY)
  • Time stamps are required for the Day Type analysis
  • Time stamps with seconds are required for some additional functionality of the module. Some loggers do not save time stamps in this format. To correct this, open the file in another data analysis software and change the format of the column to an appropriate format. (This is NOT required for the Day Type analysis)
  • Data files with a large time stamp gap (i.e., months) will take a long time to parse. If your data file takes several minutes to parse, there could be an error in your data (a single time stamp from the year 1999). You may need to manually edit in a spreadsheet/text editor software.
Click "Parse CSV Data" to upload
Multiple csv files can be uploaded using the same process
  • These files do not need matching time stamps (though if there is a large time gap, it will take longer to parse).
  • Data Visualization options will be limited (the module cannot graph columns from two different data sets in the same scatter plot).
If you have an error in your data or want to upload a fresh data set, click "Start Over" in the left screen
After data is parsed, click "Next" to move on to "Clean Data"
"Use Column" selects which data columns will be included in the "Day Type Analysis" and "Data Visualization" sections.
"Is Date" Check this box if the column is dates with time stamps or just dates alone. Dates are needed for Day Type analysis and time series plots. If you have data collected on a less-than-minute interval, you need to make sure your time stamp data format includes seconds (e.g., MM/DD/YY HH:MM:SS).
"Is Time" can be used if you have seperate date and time columns in your data file. Use "Is Date" for the date portion and "Is Time" for the time portion. If you have data collected on a less-than-minute interval, you need to make sure your time stamp data format includes seconds (e.g., MM/DD/YY HH:MM:SS).
"Column Name" is the name in the column header as it appears in the imported .csv or .xlsx file.
"Alias" allows you to change the name of the data set that will be used for display purposes in the "Day Type Analysis" and "Data Visualization" sections.
"Units" can be set that correspond to the column. This is used only for data display purposes.
Once you have cleaned your data, click "Submit" and your data will be cleaned and set for analysis. Once this is complete you can continue to "Day Type Analysis" or "Data Visualization".
