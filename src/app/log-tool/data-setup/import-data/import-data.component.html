The Data Exploration module is designed to help users establish Day Types for equipment analysis incorporating multiple operating types (e.g. Compressed Air Assessments). It also provides a robust Data Visualization component to allow users to create scatter plots and histograms of their data.

  • Import .csv or .xlsx files from your file system.
  • The Refine Data and Map Data steps will allow you to modify imported data.
  • Time stamps are required for Day Type Analysis.
    • Time stamps with seconds are required for some additional functionality of the module. Some loggers do not save time stamps in this format. To correct this, open the file in another data analysis software and change the format of the column to an appropriate format. (This is NOT required for the Day Type analysis)
    • Data files with a large time stamp gap (i.e., months) will take a long time to parse. If your data file takes several minutes to parse, there could be an error in your data (a single time stamp from the year 1999). You may need to manually edit in a spreadsheet/text editor software.
