Efficiency at Average Load
The efficiency at average operating power/load
"Estimate Efficiency" will predict the efficiency at the average load. The predicted efficiency is based on models generated by PSAT 2008 using TEFC (Totally Enclosed, Fan-Cooled) motor data from MotorMaster. The models use the following predictor variables:
  • Nominal Efficiency
  • Average Load Factor
  • Full-Load Speed
  • Rated Motor Power
  • Line Frequency
Note: If the values used to predict the efficiency are changed then the button will have to be re-clicked to update the predicted Efficiency.
Note: This calculation is only valid for 3-phase AC motors

Current at Load
Measured current draw at given load. This can be measured with a data logger, an ammeter, or calculated by MEASUR.
"Estimage Current" will predict the current at the average load using standard electrical engineering equations. The equations use the following predictor variables:
  • Rated Motor Power
  • Rated Voltage
  • Full-Load Speed
  • Average Load-Factor
  • Line Frequency
  • Estimated Nominal Efficiency
  • Full-Load Amps
Note: If the values used to predict the current are changed then the button will have to be re-clicked to update the predicted Current.
Note: This calculation is only valid for 3-phase AC motors

Power Factor at Load
Its the ratio of true power to apparent power, this can be measured using a clamp on meter or calculated via user inputs.
"Estimate Power Factor" will predict the power factor at the average load using standard electrical engineering equations. The equations use:
  • Average Load Factor
  • Rated Motor Power
  • Rated Voltage
  • Efficiency at Average Load
  • Current at Load
Note: If the values used to predict the power factor are changed then the button will have to be re-clicked to update the predicted Power Factor.
Note: This calculation is only valid for 3-phase AC motors
