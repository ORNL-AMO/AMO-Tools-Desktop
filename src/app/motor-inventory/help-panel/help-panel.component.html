CO2 Savings Help
This calculator lets the user determine carbon dioxide (CO2) emissions for given combustion fuel, biofuel, refrigerant charge, purchased gases, purchased electricity from the grid. It also helps to calculate the emissions fuel use for transportation. Select the emission source type to select the emission from combustion fuel, biofuel, refrigerant charge, purchased gases, purchased electricity from the grid. Next select the emission source and corresponding fuel type. For calculating emission from purchased electricity from the grid select Electricity under Emission Source Type.
