Efficiency at {{percentValue}}% Load
Efficiency at partial ({{percentValue}}%) load.
"Estimate Efficiency" will predict the efficiency at the ({{percentValue}}%) load. The predicted efficiency is based on models generated by PSAT 2008 using TEFC (Totally Enclosed, Fan-Cooled) motor data from MotorMaster. The models use the following predictor variables:
  • Nominal Efficiency
  • Full-Load Speed
  • Rated Motor Power
  • Line Frequency
Note: If the values used to predict the efficiency are changed then the button will have to be re-clicked to update the predicted Efficiency.
