Restore System Settings

Restoring system settings will provide options for resetting all application settings to their default values, returning all example assessments to their original state, deleting all user-created assessments/pre-assessments, and deleting all user-created materials. Selecting this button opens the menu where you can choose to reset one or all of these before-mentioned settings. If you reset only one option (e.g. resetting example assessments), it will not impact any other areas or options. This cannot be undone!
