Exhaust Gas Help

Off (Exhaust) Gas Temperature

Temperature of exhaust gases from the EAF (sometimes referred to as fourth hole). This is measured or estimated before the gases are mixed with outside air.

Units: ℉ or ℃

CO

Average value of CO percentage in exhaust gases. Note that the values vary with time during the cycle and this is average over the cycle time.

Units: %
Minimum Value: 0
Maximum Value: 100

H2

Average value of H2 percentage in exhaust gases. Note that the values vary with time during the cycle and this is average over the cycle time.

Units: %
Minimum Value: 0
Maximum Value: 100

Combustible Gas as CH4

Total volume of exhaust gases (use average value).

Units: %
Minimum Value: 0
Maximum Value: 100

Total Volumetric Flow Rate

Average value of combustible gases percentage in exhaust gases. Note that the values vary with time during the cycle and this is average over the cycle time.

Units: cfm or m3/min
Minimum Value: 0

Dust Loading

Dust loading for exhaust gases in terms of weight/SCF

Units: lb/scf or kg/Nm3
Minimum Value: 0

Savings Suggestions
  • Minimize volume by controlling make up air and air leaks in the oven
  • Use preheated combustion air
  • Reduce exhaust gas temperature by using one or more heat recovery methods
