CO2 Savings Help
This calculator lets the user determine carbon dioxide (CO2) emissions for given combustion fuel, biofuel, refrigerant charge, purchased gases, purchased electricity from the grid. It also helps to calculate the emissions fuel use for transportation. Select the emission source type to select the emission from combustion fuel, biofuel, refrigerant charge, purchased gases, purchased electricity from the grid. Next select the emission source and corresponding fuel type. For calculating emission from purchased electricity from the grid select Electricity under Emission Source Type. Energy Source
Type of fuel creating the emission (used to set Fuel Type dropdown options). Can choose from Steam & Hot Water, Natural Gas, Petroleum-based fuels (including diesel and gasoline), biomass fuels (including biodiesel and landfill gas), coal, and other fuels (including coke).
Fuel Type
Specific fuel creating the emission (set by Energy Source).
Total Emission Output Rate
The total emissions per unit of energy use. By default, this is set by the dropdown options, but is changeable. The default values are only kg of carbon dioxide (not carbon dioxide equivalent).
