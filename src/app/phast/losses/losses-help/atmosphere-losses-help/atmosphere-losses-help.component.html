Atmosphere Losses Help

Atmosphere Gas

Type of gas used as a process atmosphere within the heating equipment or reaction chamber To add a new gas, click “Add New Gas”. You can “start with existing gas” and edit the condition factor or add a completely new gas. You cannot use the name of a gas already in the database.

Specific heat of gas

Give average value of the specific heat for the temperature range of the gas inlet temperature and outlet temperature.

Units: Btu/(SCF-℉) or kJ/(Nm3-℃)
Minimum Value Allowed: 0

This value comes from the PHAST Materials Database. Its value can be changed here, however, it will not be saved in the database. If you change the dropdown menu value, then this number will change to reflect the value stored in the database. If you will be using the same adjustments to a value for multiple assessments, you may want to consider adding it as a new material by clicking "Add New ... "

Inlet Temperature

Inlet temperature (Celsius) of the process atmosphere as it enters the heating equipment or reaction chamber. Use temperature recorder/indicator reading or actual measurement.

Units: ℉ or ℃

Outlet temperature

Outlet temperature (Celsius) of the process atmosphere as it leaves the heating equipment or reaction chamber. Use temperature recorder/indicator reading or actual measurement.

Units: ℉ or ℃

Flow Rate

Flow rate of atmosphere gas, in terms of volume per hour from flow meter readings. Assumed to be at standard conditions and actual flow values given by a flow meter should be corrected for the pressure-temperature condition at the measurement point. Note that for batch furnaces the total weight should be corrected for cycle time to obtain weight per hour value.

Units: SCF/hr or Nm3
Minimum Value Allowed: 0

Correction Factor

Engineering correction factor multiplied to the result to account for a known systematic error.

Minimum Value Allowed: 0

Savings Suggestions
  • Optimize (minimize) use of process atmosphere flow rate
  • Preheat atmosphere to a higher temperature entering the furnace
  • Note: These energy saving measures are for guidance only. Not all measures are applicable under all operating conditions. There may additional measures when considering specific situations and the user is encouraged to review and apply the appropriate measures
