Atmosphere Gas
Type of gas used as a process atmosphere within the heating equipment or reaction chamber To add a new gas, click "Add New Gas". You can "start with existing gas" and edit the condition factor or add a completely new gas. You cannot use the name of a gas already in the database.
Average Specific Heat of Gas
Amount of heat per unit mass required to raise the temperature by one degree. For most materials, specific heat changes with temperature. At low temperatures, this change is negligible and it can be assumed that specific heat is constant across the temperature differential. At higher temperatures, the material's specific heat increase is more substantial. This makes heat loss or heat demand calculations difficult. As MEASUR does not take this change into account, it is recommended that an average value of specific heat for a certain temperature range is used. MEASUR provides an default value for several materials.
This value comes from the Process Heating Materials Database. Its value can be changed here, however, it will not be saved in the database. If you change the dropdown menu value, then this number will change to reflect the value stored in the database. If you will be using the same adjustments to a value for multiple assessments, you may want to consider adding it as a new material by clicking "Add New ... "
Inlet Temperature
Inlet temperature (Celsius) of the process atmosphere as it enters the heating equipment or reaction chamber. Use temperature recorder/indicator reading or actual measurement.
Outlet Temperature
Outlet temperature (Celsius) of the process atmosphere as it leaves the heating equipment or reaction chamber. Use temperature recorder/indicator reading or actual measurement.
Flow Rate
Flow rate of atmosphere gas, in terms of volume per hour from flow meter readings. Assumed to be at standard conditions and actual flow values given by a flow meter should be corrected for the pressure-temperature condition at the measurement point. Note that for batch furnaces the total weight should be corrected for cycle time to obtain weight per hour value.
Correction Factor
Engineering correction factor multiplied to the result to account for a known systematic error.
