Energy Input Help
The data entered on this page does not directly impact the calculations by the PHA for the total gross heat of the system. This page is intended as a comparison to the PHA calculation.
Natural Gas Heat Input
Total heat input to the heating system (furnace/oven) from all sources of heat supplied. They may include natural gas, carbon, fuel oil etc. as applicable. Can be calculated using "Calculate using flow rate" button. Note that for batch furnaces the total heat input should be corrected for cycle time to obtain per hour value.
Coal Carbon Injection
Mass (weight) of coal or carbon injection on a per hour basis. Note that for batch furnaces the total carbon injection should be corrected for cycle time to obtain per hour value.
Coal Heating Value
Higher heating value for the coal or carbon injected. The HHV of coal is dependent on the type and source. The default value represents the average coal HHV. Below is a table of HHV for the various grades of coal.

Electrode Use
Mass (weight) of electrode use on a per hour basis. Note that for batch furnaces the electrode use should be corrected for cycle time to obtain per hour value.
Electrode Heating Value
Higher heating value for the graphite electrode material.
Other Fuels
Heat supplied from other sources, if any. Note that for batch furnaces the total weight should be corrected for cycle time to obtain per hour value.
Electricity Input
Electricity Input into system. Note that for batch furnaces the electricity input should be corrected for cycle time to obtain per hour value.
This field is used to calculate the Exhaust Gas Losses as an alternative to utilizing the Exhaust Gas Loss page. If there is no data on the Exhaust Gas Loss page, then Exhaust Gas Loss is calculated as the difference between the Gross Heat Input (the sum of the chemical, electrical, and exothermic reaction heat inputs) and Total Net Heat Required.
