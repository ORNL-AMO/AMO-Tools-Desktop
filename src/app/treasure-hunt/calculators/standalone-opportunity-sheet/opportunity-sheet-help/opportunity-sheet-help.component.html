Custom Savings Opportunity
This calculator provides a space to add a Treasure Hunt Opportunity without using a calculator (such as after having done off-sheet calculations). Enter Baseline and Modification Utility use to calculate savings.
Engineering Services
Approximate cost of engineering services associated with implementing the opportunity.
Material Costs
Approximate cost of new materials / equipment associated with implementing opportunity.
Labor Costs
Approximate cost of labor associated with implementing the opportunity.
Other Cost Description
Provide a name to describe the additional cost.
Other Cost
Approximate addtional costs associated with implementing the opportunity.
Additional Savings Description
Provide a name to describe the additional savings.
Additional Savings
Approximate additional cost savings associated with implementing the opportunity.
Opportunity Notes
Use this field for any other notes on the opportunity during analysis or later in implementation.
Project Title
Provide a short, descriptive name for the opportunity.
Process / Equipment
Provide a short description of what would be impacted by this opportunity.
Project Owner / Lead
Provide a name for the owner of the project where the opportunity is located.
Owner / Lead
Provide the name for the business unit that will own the opportunity or provide a name for the facility or portion of the facility where the opportunity is located.
Description
Provide details about the opportunity for use during implementation.
Utility Consumption
Annual utility consumption before or after implementation of the opportunity.
Utility Type
Type of utility that this opportunity will impact.
Effort to Implement
The amount of effort the savings opportunity will take to implement on a scales of 1-10
Other Annual Savings
Approximate additional annual cost savings associated with implementing the opportunity, such as reduced maintenance and labor cost.
Provide a name for the savings and a savings amount.
