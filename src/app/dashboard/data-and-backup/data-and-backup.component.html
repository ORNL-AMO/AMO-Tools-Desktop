Data Storage
MEASUR data is stored and managed internally by the application. DOE and MEASUR support staff do not have access to your data.

Data is not accessible to users through the operating system or local machine. For greater control of application data, the MEASUR team recommends that users take advantage of the automatic backup feature (below).

Users can still create manual backups or export data as needed. These features are available in the MEASUR dashboard, as well as in the header of assessments, inventories, and treasure hunts.

MEASUR data is stored and managed internally by your browser*. DOE and MEASUR support staff do not have access to your data.

Data is unique to each browser (Chrome, Firefox, Edge), and is only accessible through exported files and backup files created within the application. For example, user data created while working in Chrome will not be reflected in Edge, nor in the desktop version of the application and vice-versa. For greater control of application data, the MEASUR team recommends that users take advantage of the automatic backup feature (below).

Users can still create manual backups or export data as needed. These features are available in the MEASUR dashboard, as well as in the header of assessments, inventories, and treasure hunts.

* NOTE: If you have enabled browser settings which disable "site data," "cookies," or similar browser storage, MEASUR may lose functionality or be unable to run entirely.

For more information on data storage and collection, please view the privacy notice
